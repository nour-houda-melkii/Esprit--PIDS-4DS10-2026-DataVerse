FX-AlphaLab – Multi-Modal FX Intelligence System

Overview
FX-AlphaLab is an intelligent multi-agent system designed to analyze the foreign exchange (FX) market by combining multiple data sources, including macroeconomic indicators, technical signals, market sentiment, and geopolitical events.
The goal is to transform complex and fragmented data into clear, explainable, and actionable trading insights. 
Features
Multi-agent architecture for FX analysis
Integration of macroeconomic, technical, sentiment, and geopolitical data
Real-time data processing and signal generation
Conflict resolution between signals
Conviction scoring for decision support
Explainable outputs for better understanding
Tech Stack

Architecture

The system is based on a multi-layer architecture:

Data Layer: Collects data from sources like MetaTrader5, Investing.com, TradingView, and news APIs
Agent Layer:
Macroeconomic Agent
Technical Analysis Agent (LSTM)
Sentiment Analysis Agent (NLP models)
Geopolitical Event Agent
Orchestration Layer: Aggregates signals, resolves conflicts, and computes conviction scores
Deployment Layer: Web dashboard for visualization and decision-making
